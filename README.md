# Online-vedio-streaming-links-
these are links that are used to stream tv shows using VLC (Video lan Media player)

how to use:
1.download vlc (https://www.videolan.org/vlc/#download)
2.run and setup vlc
3.open vlc
4.go to menu toolbar
5.scroll down to stream
6.select network stream
7.paste the link and click stream
8.go through the setting by click next
9.wait 5-10 mins for the streams to load
10.enjoy
