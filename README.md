# Online-vedio-streaming-links-
these are links that are used to stream tv shows using VLC (Video lan Media player)

how to use:
download vlc (https://www.videolan.org/vlc/#download)
run and setup vlc
open vlc
go to menu toolbar
scroll down to stream
select network stream
paste the link and click stream
go through the setting by click next
wait 5-10 mins for the streams to load
enjoy
